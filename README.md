# 🎬 棠荞票根工坊

> 智能输入 · 自动计算 · 灵活选择 · ✨ 永久免费 · 无需付费

一款纯本地、永久免费的异形电影票根生成工具。无需联网，所有数据不上传服务器，保护你的隐私。

**在线体验**：https://tangqiaostar.github.io/ticket-tool/

---

## ✨ 功能特性

### 🎫 票根设计
- **异形票根**：左右各 11 个半圆孔洞（直径 0.3cm）
- **自定义形状**：支持上传 PNG 透明背景蒙版，自由定制票根轮廓
- **高清输出**：画布 1417×709 px（12cm × 6cm @ 300 DPI），一键下载 PNG

### 🖼️ 背景与海报
- **背景图**：上传任意图片，中心缩放（cover）不挤压，支持模糊调节（0-20px）
- **纯色背景**：无图片时显示纯色，颜色可自定义
- **电影海报**：固定尺寸 977×639，居中裁剪填充

### 📝 右侧信息区
精确位置（左 1095px，上 36px，宽 301px，高 639px），包含以下字段：

| 字段 | 说明 |
|------|------|
| 电影片名 | 手动输入，年份自动显示为下标 |
| 上映年份 | 支持「选择年份」（1900-当前+10）或「手动输入」 |
| 外文名 | 手动输入 |
| 国家 | 手动输入 |
| 导演 | 手动输入 |
| 主演 | 手动输入 |
| 时长（分钟） | 数字输入，自动计算结束时间 |
| 上映日期 | 日历选择，显示为 日/月/年 格式 |
| 影院名称 | 手动输入 |
| 观影日期 | 日历选择 |
| 观影时间 | 起始时间 + 结束时间（根据时长自动计算） |
| 影厅 | 手动输入 |
| 座位 | 支持「选择座位」（行/列：数字+字母混排）或「手动输入」 |
| 票价 | 支持下拉建议（30-150元）或手动输入两位小数 |
| 票号 | 自动生成（外文名 + 观影日期），可手动修改 |

### 🎨 个性化定制
- **水印**：独立颜色 + 透明度调节（0-1），默认「本票根由🎬 棠荞票根工坊生成」
- **字体**：内置「霞鹜文楷屏幕阅读版」，支持上传自定义字体（.ttf/.otf/.woff）
- **字体颜色**：主文字颜色任意选择
- **上传控件**：全部配有「清除」按钮

### 🔒 隐私与版权
- **纯本地运行**：所有数据在浏览器本地处理，不上传任何服务器
- **永久免费**：无任何付费墙，无套路
- **版权保护**：© 2026 棠荞，采用 CC BY-NC 4.0 许可证，禁止商用
